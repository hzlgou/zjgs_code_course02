

## 作业1（必做）：
### 模拟登录界面
界面上放置两个文本输入框（EditText），用于输入用户名和密码。一个按钮（Button），一个文本显示控件（TextView）。

### 用户信息
假设有两名用户
- 用户名mike，密码m12345
- 用户名jack，密码12345

### 功能逻辑
如果用户名密码输入正确，按登录按钮后，下方的TextView上显示：欢迎您，xxx（xxx为用户名）。
如果用户名密码不匹配，按登录按钮后，下方的TextView上显示：用户名或密码错误。

### 编程提示（Java的字符串比较）：
```java
String s;
s.equals("abc"); // 把s字符串的内容和"abc"作相等与否的比较，相等返回true，否者是返回false。
s == "abc"; // 地址比较，不适用于本例。
```

## 作业2（选做）：
界面上放置一个文本输入框（EditText），一个按钮（Button），一个文本显示控件（TextView）。

TextView宽度设宽一点，把android:layout_width设为match_parent（与父容器即屏幕同宽），或300dp左右；高度200dp左右。

在文本输入框里，输入一个正整数n。点击按钮后，在TextView中显示出n以内的素数。

例如：n=10
显示：2，3，5，7
